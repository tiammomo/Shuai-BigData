# Apache Kylin 中文文档

> Kylin 是一个开源的分布式分析引擎，提供 Hadoop/Spark 上的 SQL 查询及多维分析能力。

## 文档列表

| 文档 | 说明 |
|------|------|
| [01-architecture.md](01-architecture.md) | 架构详解 |

## 快速入门

```java
// Maven 依赖
<dependency>
    <groupId>org.apache.kylin</groupId>
    <artifactId>kylin-jdbc</artifactId>
    <version>4.0.3</version>
</dependency>
```
