# Apache Druid 中文文档

> Druid 是一个高性能的实时分析数据库，支持高并发 OLAP 查询。

## 文档列表

| 文档 | 说明 |
|------|------|
| [01-architecture.md](01-architecture.md) | 架构详解 |

## 快速入门

```java
// Maven 依赖
<dependency>
    <groupId>org.apache.druid</groupId>
    <artifactId>druid-sql</artifactId>
    <version>26.0.0</version>
</dependency>
```
