# Presto/Trino 中文文档

> Presto/Trino 是一个分布式 SQL 查询引擎，支持跨数据源联邦查询。

| 文档 | 说明 |
|------|------|

## 快速入门

```java
// Maven 依赖
<dependency>
    <groupId>io.trino</groupId>
    <artifactId>trino-jdbc</artifactId>
    <version>419</version>
</dependency>
```

## 核心概念

```
┌─────────────────────────────────────────────────────────────────┐
│                     Presto/Trino 核心架构                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │

│  │                      Coordinator                         │   │
│  │  - SQL 解析和计划                                        │   │
│  │  - 任务调度                                              │   │
│  │  - 结果合并                                              │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                      Worker                              │   │
│  │  - 实际数据处理                                          │   │

│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                      Connector                           │   │
│  │  - 数据源适配器                                          │   │
│  │  - 元数据访问                                            │   │

│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
