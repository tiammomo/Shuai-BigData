## 快速入门

```java
// Maven 依赖
<dependency>
    <groupId>org.apache.pulsar</groupId>
    <artifactId>pulsar-client</artifactId>
    <version>3.1.0</version>
</dependency>
```

## 核心概念

```
┌─────────────────────────────────────────────────────────────────┐
│                     Pulsar 核心架构                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                      Broker                              │   │
│  │  - 无状态服务，消息分发                                   │   │
│  │  - 负责消息的接收和投递                                   │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                      BookKeeper                          │   │
│  │  - 有状态存储，消息持久化                                 │   │
│  │  - 多副本保证数据可靠性                                   │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                      ZooKeeper                           │   │
│  │  - 元数据存储                                            │   │
│  │  - 集群配置管理                                          │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                      Tenant / Namespace / Topic          │   │
│  │  - 多租户支持                                            │   │
│  │  - 层级结构管理                                          │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
